AR Smartsheet Automation – Step Runner (HTML)

Local helper UI for colleagues without programming background. No server install needed.
Smartsheet + Code generation steps can run here. ArcGIS / arcpy steps must be executed in ArcGIS Pro (not possible inside browser).
Important: This page calls Smartsheet REST API directly. If your browser blocks CORS or your organization restricts it, calls may fail.

Values are injected from .env via build_config.py. To change token or sheet IDs, edit .env and re-run the build script. No credentials are typed into the browser.
